# E-Commerce Sales Analysis  
### Task 01 | Data Science Internship – Future Intern  

#Project Overview: 
This project analyzes real-world e-commerce sales data to identify key business insights such as:  
- Which products and categories generate the most revenue?  
- When sales peak during the year?
- Which regions contribute the most profit?  

The goal is to support better business decisions through data analysis and visualization.

---

##  Key Insights  
- **Technology** category generated the highest sales.  
- **Phones** were the top-selling sub-category.  
- **November** had the highest monthly sales.  
- **West region (California)** contributed the most profit.
